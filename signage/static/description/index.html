Professional showcase administration

When using Odoo-Signage, it will be possible to control multiple
screens from remote location.


Using Open Source software, we have created a Raspberry Pi that can can be crontrolled from a distance. You can easily connect as many as Raspberries as you desire.

We have used software comming from Screenly.io to manage the run any showcase from a distance. We use Puppet to ramp up the scaling to include all locations. But this is all optional and beyond the information management at large.

When running this software, it's possible to publish and manage data on a screen in a distance. It can be photos, moving animations or films and it can be links to active webpages.

Master information on external screens from remote.

  • Photos.

  • Live webgages.
  • CMS / database connectivity.

1. Set Token

Inside Odoo, you can edit the Token. Here it shows where that is. In this case, Token is set to "123".

You shall use the token in the URL as a "secret key" to access your dynamic data. It will help you to keep the data private.

2. Show content

When setting up a site or a service, the default page will show. In this view, you can edit the content by clicking "edit" in the very right corner of the page.

3. Edit content

To edit content, simply press "Edit" in the upper right corner of the sreen. Drag-and-drop from the template to the left. Edit text and images as desire.

4. Mission complete

When finish, use the Token in the URL to see the result. Each time you relaod the page, every page as you've created will show, as part of the endless loop.

Raspberry Pi & Screenly

There are many options out there and this is one of them.
For inspiration purpose only!

In this particular set-up, we've used a small Raspberry Pi computer connected to a monitor, screen or a television.

The Raspberry Pi serves as the Internet server and the software Screenly is running to manage the presentation.

The user interface is very initutive and quite self-explaining. With the right knowledge it's quite easy to get started in a short period of time. It's easy when you know how!
